My Projects

Social Media Platform Prototype

A fully functional social media platform prototype developed using object-oriented principles in Java, with a GUI built in Java Swing. It features a sign-in/login system, allowing users to follow others and gain followers. Users can also create posts with both text and images, and engage with content through likes and comments. The design emphasizes usability, making it a strong foundation for further development.

Quizzer

Quizzer is a command-line program written in C designed for teachers and students. Teachers can create quizzes that students can take either as practice or as formal tests. In practice mode, students receive immediate feedback on incorrect answers and have the opportunity to try again. In test mode, students are allowed only one attempt per question, and their scores are automatically calculated. The program focuses on functionality and simplicity, making it a straightforward tool for educational purposes.

House Automation System

This house automation system was made using C++ within the Arduino IDE and built on a physical framework powered by an Arduino Uno. The setup includes a bluetooth module, a battery, wiring, and other essential components. Key features of the system include a rain-detection mechanism that automatically closes a window upon rain detection via bluetooth, a light bulb that can be controlled through an Arduino app, and a door that can be opened or closed using the same app. The project showcases a practical and efficient implementation of home automation technology.

Landing Page

This landing page is crafted using HTML5, CSS, and JavaScript, designed to be fully responsive, ensuring optimal display across devices. It features interactive elements that enhance user engagement and create a dynamic user experience. Additionally, Google Analytics is integrated into the page to provide valuable insights into user behavior and site performance. The combination of clean code and modern design principles makes this landing page both visually appealing and functionally robust.
